DNS Filtering Lists



These are my custom filtering lists I use for AdGuard.
I use AdGuard Home, however these can also be used with AdGuard DNS.

General Blocker

My personal blocker that blocks sites that haven't been blocked by most other filters.



Anti Censorship Blocker

Pulls an uno reverse on blockers, by blocking them.
Mainly school blockers and their trackers.



Adult Blocker

Blocks adult content.



Installation Instructions

  1. Copy the list's URL
  2. Navigate to your GUI panel
  3. Go to DNS blocklists
  4. Add New List
  5. Add Custom List
  6. Give list a name
  7. Paste URL
  8. Save
