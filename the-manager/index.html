The Manager

What is it?

The Manager is a Discord bot that gives several utilities, moderation commands, and other fun commands to help make your Discord server life easier.

What's good about it?

With powerful features, it allows you to empower your staff and yourself, making server management easier.

How so?

For example, you can reply to a message and use purge after to tell the bot to purge X messages after the message you've replied to.
